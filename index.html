Text Styles

H1 - Main page heading

  • Font family: Candara
  • Font size: 50 pixels
  • Font weight: bold (default setting)

H2 - Section headings

  • Font family: Candara
  • Font size: 30 pixels
  • Font weight: bold (default setting)

H3 - Subheadings

  • Font family: Candara
  • Font size: 20 pixels
  • Font weight: bold (default setting)

P - Paragraph text

  • Font family: Arial
  • Font size: default
  • Font weight: normal (default setting)
